## 如何成为贡献者？

我们欢迎任何人参与贡献！以下是一些贡献方式：

1. **报告 Bug**：在 GitHub Issues 中提交问题
2. **提交代码**：通过 Pull Request 贡献代码
3. **改进文档**：帮助完善项目文档
4. **功能建议**：提出新功能想法
5. **推广项目**：在社交媒体分享 Penshot

> 所有贡献者将在此页面获得署名！
